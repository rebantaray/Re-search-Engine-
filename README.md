2.2 How was this project created? The creation of the Re-search Engine was a journey of technical persistence.

The Foundation: The project began by revisiting the fundamentals of HTML (HyperText Markup Language) found in Class 6 and 7 Computer Science books at Don Bosco Park Circus.

The Design Logic: I utilized CSS3 variables and Flexbox/Grid layouts to create a "futuristic hub" aesthetic. I implemented Glassmorphism (using backdrop-filter: blur) to give the site a professional, high-end feel.

The Search Algorithm: Using Vanilla JavaScript, I developed a keyword-detection engine. This script sanitizes user input and intelligently routes the user to specific educational sub-pages, making the experience feel like a real-time AI search hub.
